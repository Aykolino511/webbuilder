Das Trio des Webs verstehen

Jede moderne Website besteht aus drei Grundbausteinen.
Entdecke interaktiv, wie HTML, CSS und JavaScript zusammenarbeiten.

HTML

Die Struktur. Das Skelett der Seite. Es definiert, was auf der Seite ist (Texte, Bilder, Buttons).

CSS

Das Design. Die Kleidung der Seite. Es definiert, wie es aussieht (Farben, Layout, Schriftarten).

JavaScript

Die Funktion. Das Gehirn. Es definiert, was passiert, wenn man interagiert (Klicks, Animationen).

Das Live-Labor

Schalte CSS und JavaScript ein und aus, um den Unterschied zu sehen.

Steuerung

1. HTML (Struktur) Immer aktiv (Fundament)
2. CSS (Design) Deaktiviert
3. JavaScript (Logik) Deaktiviert
